🎧 TP Psychoacoustique : Protocole de Test Étudiant

🚀 CLIQUEZ ICI POUR LANCER L'APPLICATION DE TEST 🚀

Bienvenue dans ce protocole de test d'audiométrie. L'objectif de cette session est de mesurer vos seuils de perception et de discrimination sur différentes voyelles.

L'application web est entièrement autonome. Vos résultats seront automatiquement envoyés au professeur à la fin de chaque test.

⚠️ PRÉREQUIS INDISPENSABLES :

Installez-vous dans un endroit calme, sans bruit de fond.

Mettez un casque audio ou des écouteurs (pas de haut-parleurs d'ordinateur).

Ne fermez pas la page web avant l'apparition du message confirmant la fin d'un test.

ÉTAPE 1 : Préparation et Réglage du Volume (À faire une seule fois !)

Avant de commencer, vous devez régler le volume de votre ordinateur pour qu'il soit confortable, puis ne plus y toucher jusqu'à la toute fin de l'expérience.

Ouvrez l'application via le lien en haut de cette page.

Dans l'encadré 1. Patient, renseignez obligatoirement votre Nom, Prénom et Âge.

Dans l'encadré 2. Masquage & Volume, cliquez sur le bouton violet "🔊 Tester le volume".

Un bruit de souffle va être émis en continu. Ajustez le volume global de votre ordinateur (Windows/Mac) pour que ce son soit clair mais confortable (il ne doit pas faire mal aux oreilles).

Re-cliquez sur le bouton "⏹️ Arrêter le son".

🚫 Règle d'or : À partir de maintenant, vous ne devez plus du tout modifier le volume de votre ordinateur ni de votre casque.

ÉTAPE 2 : Test d'Identification Initial

Nous allons commencer par vérifier votre capacité à identifier les sons.

Dans 3. Tâche à réaliser, choisissez "3 - Identification".

Cliquez sur le gros bouton vert "LANCER LE TEST" en bas.

Un son va être joué. Cliquez sur le bouton correspondant à la voyelle que vous pensez avoir entendue. Le bouton clignotera en vert (correct) ou rouge (incorrect).

Continuez jusqu'à ce qu'une boîte de dialogue vous indique : "Test terminé".
(Vos données sont alors sauvegardées et envoyées automatiquement).

ÉTAPE 3 : Série de Détection et Discrimination (Ordre libre)

Vous devez maintenant réaliser 10 tests courts.
Pour éviter les effets de fatigue ou d'apprentissage, ne les faites pas dans l'ordre de cette liste. Mélangez-les ! (Par exemple : faites une détection, puis deux discriminations, puis une détection, etc.).

À chaque fois, choisissez le test dans le menu déroulant "Tâche à réaliser", vérifiez vos sons dans "Son Standard / Déviant", et cliquez sur "LANCER LE TEST".

A. Les 4 tests de DÉTECTION (Tâche 1)

Sélectionnez "1 - Détection". Pour chaque test, choisissez la voyelle cible dans le menu "Son Standard", puis lancez le test. Cliquez sur le gros bouton "ENTENDU" dès que vous percevez le son.

[ ] Détection de la voyelle i

[ ] Détection de la voyelle o

[ ] Détection de la voyelle ou

[ ] Détection de la voyelle u

B. Les 6 tests de DISCRIMINATION (Tâche 2)

Sélectionnez "2 - Discrimination". Dans cette tâche, vous entendrez 3 sons à la suite. L'un d'eux est différent des deux autres. Cliquez sur 1, 2 ou 3 pour indiquer la position de l'intrus.
Attention à bien configurer le "Son Standard" et le "Son Déviant" avant de lancer :

[ ] Standard : o | Déviant : i

[ ] Standard : o | Déviant : ou

[ ] Standard : o | Déviant : u

[ ] Standard : i | Déviant : u

[ ] Standard : ou | Déviant : i

[ ] Standard : ou | Déviant : u

(Rappel : attendez bien le message "Test terminé" à la fin de chaque sous-test !)

ÉTAPE 4 : Test d'Identification Final

Pour clôturer l'expérience, nous refaisons exactement le même test qu'au tout début.

Dans 3. Tâche à réaliser, choisissez à nouveau "3 - Identification".

Cliquez sur "LANCER LE TEST" et répondez jusqu'à la fin.

🎉 L'expérience officielle est terminée, merci pour votre participation !

P.S. : Si la psychoacoustique vous amuse, l'application est à vous ! Sentez-vous totalement libres de tester d'autres paires de discrimination (ex: i-o, u-ou) ou de refaire des identifications pour voir si vos propres seuils évoluent avec l'entraînement. Toutes les données supplémentaires sont les bienvenues ! ;)
